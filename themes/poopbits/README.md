# GNU Elefantoj

A minimal Hugo theme based on [Elephants](https://gitlab.com/meibenny/elephants)

## Features
* High Contrast
* Syntax highlighting
* Taxonomy lists

## Installation
In the folder of your Hugo site, run:
```
    git submodule add https://gitlab.com/juancolacelli/elefantoj.git themes/elefantoj
```
